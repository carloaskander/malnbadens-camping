# Malnbadens Camping Website

A modern, multilingual camping website built with React and Vite for Malnbadens Camping in Hudiksvall, Sweden.

## 🏕️ Features

- **6 Languages**: Swedish, English, Norwegian, German, French, Finnish
- **Responsive Design**: Mobile-first approach with Material-UI
- **Accommodation Booking**: Camping, cottages, and hostel options
- **SEO Optimized**: Meta tags, structured data, and sitemap
- **Modern Performance**: Vite build system with optimized assets

## 🚀 Tech Stack

- React 18 + Vite
- Material-UI v5 with custom theming
- React Router v6 with internationalization
- i18next for translations
- Framer Motion for animations
- Contentful CMS integration
- Vercel deployment
